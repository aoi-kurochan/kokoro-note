# 心ノート（こころノート）

15の問いに答えるだけの、やさしいジャーナリング（日記）Webアプリです。
HTML1枚で動き、スマホ・PCのブラウザでそのまま使えます。

- 今日の「心の天気（晴れ・くもり・雨・嵐）」と心スコアを表示
- まず気持ちを受け止めて、そっと前を向かせる励ましメッセージ
- 過去7日間の心もよう／日付ごとの振り返り
- もっと相談したい人向けに、AIに貼れる相談文を自動生成

## プライバシーについて
書いた内容は、使った人自身の端末（ブラウザ）の中だけに保存されます（localStorage）。
インターネットやサーバーには一切送信されません。作者や他の人が、その内容を見ることはできません。
